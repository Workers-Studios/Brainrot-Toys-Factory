# Brainrot-Toys-Factory
[Tycoon] Jeu Roblox

# Description
Le concept central du jeu repose sur la construction et l'optimisation d'une usine de production. Le joueur commence par acquérir son premier spawner et convoyeur pour générer des Brainrot Toys. Ces derniers sont ensuite transformés en Brainrot portable et déposés dans l'usine, créant ainsi un système de génération de revenus automatique et évolutif.
